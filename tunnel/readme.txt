Il s'agit ici de créer un tunnel HTTP. Le langage choisi est python.

Il y aura deux scripts python à écrire : un script "serveur" coté "maison" et un script "client" coté "bureau".

Le script launch_proxy permet de créer un proxy local dans une machine virtuelle. Pour l'instant, il est testé et fonctionnel uniquement sur pcmt4.
